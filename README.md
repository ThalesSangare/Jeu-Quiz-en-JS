Mini Quiz JavaScript
Description

Ce projet est un quiz interactif développé en HTML, CSS et JavaScript.
Il permet à l’utilisateur de répondre à plusieurs questions, de soumettre ses réponses, et d’obtenir un score.
Les bonnes réponses sont mises en évidence en vert, et les mauvaises en rouge.
Le projet est structuré pour être simple, modulable et facile à étendre avec de nouvelles questions.

Fonctionnalités

Création dynamique des questions et réponses via JavaScript.

Boutons radio pour sélectionner une seule réponse par question.

Bouton "Soumettre" pour calculer et afficher le score.

Mise en évidence visuelle des bonnes et mauvaises réponses.

Code structuré pour faciliter l’ajout de nouvelles questions.

###############################################
Utilisation

Lire chaque question et sélectionner une réponse.

Cliquer sur le bouton Soumettre.

Le score total s’affiche et les réponses sont colorées selon leur correction.


###############################################
Personnalisation

Ajouter de nouvelles questions :

Modifier le tableau bonnesReponses dans script.js.

Ajouter un appel à creerQuestion() avec le titre et les réponses.

Modifier le style des réponses correctes et incorrectes dans style.css pour personnaliser l’apparence.
